High-Precision Number: BigInteger and BigDecimal

  • You can do anything with a BigInteger or BigDecimal that you can with an int or float, but use method calls instead of operators.

  • BigInteger supports arbitrary-precision(任意精度的) integers

  • BigDecimal is for arbitrary-precision fixed-point numbers(定点数)

Array in Java is safe

  • Range: Can not be accessed outside of its range.
  • Initialized: When you create an array object,in fact,you create a reference which is initialized to be null.
  • Why safe? As references must be pointed to objects,when you use a reference which is still null,problem will be reported in the runtime.
